電気公式学習ツール (Electricity Formula Tool)
このプロジェクトは、物理学や電気工学で頻出する公式を、ブラウザ上で直感的に確認・学習するためのWebアプリケーションです。

特徴
動的な公式表示: セレクトボックスから項目を選ぶだけで、関連する公式と解説が即座に表示されます。

美しい数式レンダリング: MathJax を採用しており、LaTeX形式で記述された複雑な物理数式を綺麗に表示します。

視覚的理解: クーロンの法則やキルヒホッフの法則など、公式の背後にある原理や関係性についても簡潔な解説を加えています。

構成
Electricity-Formula-Tool.html: アプリケーションのUI構造を定義。

Electricity-Formula-Tool.css: スタイル調整用（必要に応じて作成してください）。

Electricity-Formula-Tool.js: 公式データと表示制御ロジックを管理。

含まれる公式
オームの法則: 電圧・抵抗・電流の関係。

合成抵抗: 直列・並列接続および和分の積。

抵抗率: 物質固有の抵抗と導電率の関係。

電流と回路: 電流の定義、分圧・分流の法則。

静電気: クーロンの法則、電場、電位。

キルヒホッフの法則: 第1法則（電流則）、第2法則（電圧則）。

導入方法
HTMLファイルの <head> セクション内に MathJax の CDN を読み込んでいることを確認してください。

ブラウザで Electricity-Formula-Tool.html を開くだけで利用可能です。
